# Dexterous Hand movements — free sample (`bulk_9_videos_933d5ad5`)

Public **preview** of a **Quality Vision Motion Dataset Engine** export: **MediaPipe Hands** (21 landmarks), **wrist-relative** fields (`wrist_position`, `wrist_relative_keypoints`), **dexterous** analytics (`finger_angles_deg`, `grip_type_proxy`, hand **motion_intelligence** v3), and the same `dataset/` layout as commercial JSONL bundles.

This folder is intended as a **high-quality free sample** for robotics and manipulation pilots (e.g. evaluation by **AgiBot**, **Unitree**, **UBTECH**, and similar teams). It refines earlier public hand samples with stronger metrics and wrist-centric post-processing.

- **Start here:** [`dataset/ONEPAGER.md`](./dataset/ONEPAGER.md) — metrics, processing, deliverables, and quality definitions for this job.
- **Merged stream:** [`dataset/data.jsonl`](./dataset/data.jsonl)
- **Per-source splits:** [`dataset/per_video/`](./dataset/per_video/)
- **Use cases:** [`dataset/use_cases.md`](./dataset/use_cases.md) — dexterous / manipulation focus (not locomotion).

Raw source videos are **not** included in git (JSONL + metadata only).
