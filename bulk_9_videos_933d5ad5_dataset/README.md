# Dexterous Hand movements — free sample (`bulk_9_videos_933d5ad5`)

Public **preview** of a **Quality Vision Motion Dataset Engine** export: **MediaPipe Hands** (21 landmarks), **wrist-relative** fields (`wrist_position`, `wrist_relative_keypoints`), optional **dexterous** analytics (`finger_angles_deg`, `grip_type_proxy`, hand **motion_intelligence** v3), and the same `dataset/` layout as commercial JSONL bundles.

- **Start here:** [`dataset/ONEPAGER.md`](./dataset/ONEPAGER.md) — metrics, processing, deliverables, and quality definitions for this job.
- **Merged stream:** [`dataset/data.jsonl`](./dataset/data.jsonl)
- **Per-source splits:** [`dataset/per_video/`](./dataset/per_video/)

Raw source videos are **not** included in git (JSONL + metadata only).  
For **larger commercial batches** (e.g. **~140k+ HQ frames**, multiple clips segmented for **specific dexterous motions**), see [Dataset pricing](https://qvision.space/dataset-pricing) and contact **info@qvision.space**.
